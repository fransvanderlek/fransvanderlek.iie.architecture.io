RAMI4.0 : Architecture View - Presents Factory Manifest ()
This view shows how the end-to-end process works in workplace-1, including all human, machine and software (communication) steps. One key aspect is that the conveyor belt is triggered via software as a result of the robot arm sending a notification message. There is no physical or human intervention. The advantage of a software implementation is that we can readily enrich the trigger with data about the object itself.
